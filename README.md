
# Flight Reservation System

The Flight Reservation System is a Java program that allows users to manage flights, book tickets, and view flight availability. It provides a simple and efficient solution for travel planning.

## Features

- View available flights
- Book tickets for a specific flight
- Display flight details including source, destination, seats available, and price
- User-friendly interface for easy navigation

## Usage

1. Compile the Java source code: `javac FlightReservation.java`
2. Run the program: `java FlightReservation`
3. Choose options from the menu to view flights or book tickets
4. Follow the on-screen instructions to complete the booking process

## Requirements

- Java Development Kit (JDK)
